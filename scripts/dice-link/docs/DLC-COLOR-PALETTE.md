# DLC Color Palette

Official color system from Foundry Dice Link module (DLC). All colors in DLA should conform to this palette.

## Core Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Brand | #6f2e9a | Main accent, primary UI elements |
| Dark Background | #1a1a2e | Main background |
| Card Background | #16213e | Cards, containers |
| Border | #0f3460 | Borders, dividers |
| Text Primary | #e0e0e0 | Main text |
| Text Secondary | #a0a0a0 | Secondary text, labels |

## Mode Indicators

| Mode | Color | Hex | Usage |
|------|-------|-----|-------|
| Digital | Indigo | #6366f1 | Player in digital mode |
| Manual | Green | #10b981 | Player in manual mode |
| Pending | Gray | #D5D5D6 | Player in pending state |

## Accent Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Light Purple (Gradient/Accent) | #a78bfa | Gradient accents, hover states, highlights, GM indicator |
| Success | #10b981 | Success states, positive actions |
| Warning | #f59e0b | Warning states, caution |
| Danger | #ef4444 | Error states, destructive actions |
| Blue Light | #5d9eca | Alternative accent, info states |

## Special Uses

- **Roll Window Header:** Uses light purple accent for headings
- **Roll Result Values:** Uses light purple accent for prominent numbers
- **GM Indicator:** Uses light purple (#a78bfa) for visibility
- **Gradients:** Combine primary brand color with light purple
